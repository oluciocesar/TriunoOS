# Banco de imagens

> Acervo de imagens **reais** do negócio: fotos profissionais do dono (quando
> ele é a cara do negócio), fotos de produto, de equipe, da empresa, bastidores
> — qualquer imagem própria que possa ser usada nos materiais (carrossel, post,
> proposta, slide).
>
> As skills de conteúdo consultam esta pasta **ANTES** de partir pra imagem
> gerada por IA, priorizando sempre foto real quando existir.

---

## Como adicionar imagens

Duas formas — a primeira é a mais fácil:

1. **Pelo chat (recomendado):** arrasta as imagens direto na conversa com o
   Claude. Ele salva aqui já renomeadas na ordem e descreve cada uma no índice.
   Você não precisa fazer mais nada.
2. **Manual:** arrasta os arquivos direto pra esta pasta no seu computador:
   `identidade/banco-imagens/`. Depois é só avisar o Claude que ele organiza a
   numeração e atualiza o índice.

---

## Regra de nomeação (importante)

Toda imagem é renomeada com **número sequencial**, na ordem em que entra no
banco: `1.jpg`, `2.jpg`, `3.png`...

- A **extensão original é mantida** (jpg, png, webp...).
- Imagens novas continuam a partir do **maior número já usado** no banco
  (se já existem 10, as próximas são 11, 12, 13...).
- **Nunca reaproveitar** número de imagem apagada — a contagem só sobe.
